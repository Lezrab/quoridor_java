Comment lancer le jeu ?

  - Pour cela, se placer dans le dossier "Quoridor/ws"
  - Exécuter la commande : "ant all"
  - Les classes sont compilées, le MANIFEST ajouté, le jar créé, la javadoc générée.
  - Lancer Quoridor.jar avec la commande "java -jar Quoridor.jar" du dossier ws (veillez à ne pas le changer d'endroit).
